GIT WORKFLOW FOR EACH PERSON
─────────────────────────────────────────────────────────────────────────────

EVERY PERSON FOLLOWS THIS:

1️⃣  MORNING - Sync with team
    git checkout main
    git pull origin main

2️⃣  CREATE - Your feature branch
    git checkout -b feature/<your-module>/<feature-name>

3️⃣  WORK - Make changes ONLY in team/<your-module>/

4️⃣  COMMIT - Frequently throughout the day
    git add team/<your-module>/
    git commit -m "feat(<module>): description"

5️⃣  PUSH - When feature is ready
    git push origin feature/<your-module>/<feature-name>

6️⃣  REVIEW - Create PR on GitHub
    - Go to GitHub
    - Click "New Pull Request"
    - Request reviews from team
    - Wait for approval
    - Merge when approved
